# 🛠️ Create: Extra Recipes

A lightweight mod that expands crafting possibilities for **Vanilla Minecraft** and improves integration with the [Create](https://modrinth.com/mod/create) ecosystem.

This mod introduces **new crafting recipes, progression tweaks, and automation-friendly mechanics**, while staying faithful to the vanilla gameplay experience.

The goal is simple:
make items **more accessible, more logical, and automation-friendly**.

## ✨ Vanilla Tweaks

Several vanilla recipes have been adjusted or added to improve progression and accessibility.

### 🔁 Recipe Improvements

- Comparators and Repeaters can now be crafted using other stone variants instead of only regular stone.
- Raw Ore Blocks can be smelted directly to obtain their corresponding metal blocks.
- Smithing Templates are now cheaper to duplicate.

![Smithing Template Cheaper](https://cdn.modrinth.com/data/cached_images/3418ee482a6d6fda72a98437f252eb760bbcf7d6.png)

### 🧰 New Craftable Items

Several previously uncraftable items now have recipes.

Examples include:

- Nautilus Shell
- Horse Armor
- Trial Keys
- **Various rare or exploration-based items**

![Trial Key](https://cdn.modrinth.com/data/cached_images/b54ea6cd44c56b4275b875a766f186e4a390ded4.png)

## 🏭 Automation Recipes

Several new **Create-compatible processing chains** have been introduced.

These recipes allow certain rare resources to be obtained through factory automation.

Examples include:

- Processing Nether materials
- Alternative Diamond production
- Additional crushing and processing recipes

![Crushing Nether Bricks](https://cdn.modrinth.com/data/cached_images/adb502dcef2877dbf740eb3edd3971731d6285d1.png)
![Diamond](https://cdn.modrinth.com/data/cached_images/d663b1ac1b22507549ff3a62e8a215e80286fa23.png)

## 🟧 Copper Oxidation

Copper blocks can now **oxidize using water**, allowing easier automation of the copper aging stages.
Perfect for builders who use **oxidized copper palettes**.
